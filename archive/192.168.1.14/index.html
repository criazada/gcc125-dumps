Descrição do trabalho

     Nome servidor: www.aluno.ufla.br
      IP: 192.68.1.14
      Nome Cliente: aluno
      IP:192.168.1.15
     Inicialmente foi acessada a VM por meio do comando ssh aluno@192.168.1.14 , que foi configurado como servidor e o ip 192.168.1.15 foi configurado como clientet. Para configurar o horário foi utilizado o serviço de hora do projeto NTP.br. Para o cliente foi utilizado o NTP data juntamente com o id do servidor, para que fosse realizada a configuração do horário em relação ao servidor.
     Por seguinte, foi realizada a instalação do servidor wed no ip 192.168.1.14, para isso foi necessário instalar o apache2 e criar um novo arquivo de configuração ( sudo nano cp 000-default.conf aluno.conf) . Logo em seguida foi alterado o nome do ServerName para www.aluno.ufla.br e acrescentado ServerAlias aluno.ufla.br e DocumentRoot /var/www/aluno e para finalizar foi criado uma nova pasta para o domínio – sudo mkdir /var/www/aluno. Feito isso foi criada uma página index.html.
     Por fim foi adicionado o arquivo de configuração na lista de arquivos a serem carregados pelo Apache e realizado o acesso ao site.
